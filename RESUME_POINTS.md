# Resume Points For Fraud Detection System

**Fraud Detection System | Python, FastAPI, MySQL**

- Developed a real-time fraud detection system to process and monitor 10,000+ financial transactions per day using FastAPI and MySQL.

The system accepts transaction data, evaluates fraud risk in real time, stores results in MySQL, and exposes analytics endpoints for monitoring suspicious activity. I also documented the architecture, testing plan, and deployment process as part of a final-year style project submission.
